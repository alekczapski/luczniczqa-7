Selenium WebDriver 🤖

( bindings, protocol, standalone server, proxy-like )

  • a collection of language specific bindings to drive a browser (https://www.seleniumhq.org/)
  • interact with the HTML elements within a page

  • RESTful web service using JSON over HTTP
  • JSON Wire protocol (Selenium) -> W3C WebDriver protocol (On June 5, 2018 WebDriver moved to a W3C recommendation)

  • Open Source

How it works? 🧙

  • bindings <=REST=> WebDriver <=protocol=> browser
  • 🚖 analogy (found on the internet)

Interaction 🤼

( attributes and methods() )
  • with browser - current url, title, get_url(), back(), refresh(), switch_to_alert()...
  • with HTML elements - locating (by name(), link_text(), class(), tag()...), text, tag_name, click(), clear(), send_keys()...
  • with alert dialogs - text, accept(), dismiss()...
  • waits - Explicit (wait for a certain condition to occur, e.g. title_contains), Implicit
